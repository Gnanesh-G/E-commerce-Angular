About the Project

Explore our user-friendly E-commerce website, powered by Angular. Discover a wide range of products sourced from the API to household essentials. Our website provides a seamless shopping experience with essential features:

  • Secure User Authentication: Log in with confidence thanks to our robust validation system.
  • Effortless Registration: Create your account quickly and securely to enjoy personalized shopping.
  • Shop & Cart: Browse products and add them to your cart, visible in the navbar for easy access.
  • Efficient Checkout: Review and finalize your purchases with a single click.

Technolgy Used:

  • Angular Framework - Version 18.8.0
    • HTML - Version 5
    • CSS - Version 3
    • TypeScript - Version 5.2.2

Challenges:

Our journey in building this website was marked by several challenges, including:

  • Real-time Products: Displaying up-to-the-minute product data from the Fake Store API.
  • User Security: Implementing strong security and validation for user login and registration.
  • Cart: Creating an intuitive and secure shopping cart and checkout process for a seamless user experience.
